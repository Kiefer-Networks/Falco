Falco — Android-клиент сервисов Hetzner Online, уважающий приватность.

Управляйте всеми четырьмя API Hetzner из одного безопасного приложения:

• Hetzner Cloud — включайте, выключайте, перезагружайте, сбрасывайте, делайте снапшоты и переводите серверы в режим восстановления; управляйте томами, сетями, файрволами, плавающими IP и балансировщиками нагрузки.
• Hetzner Robot — выделенные серверы с reset, режимом восстановления и Wake-on-LAN; полная поддержка Storage Box: снапшоты, субаккаунты и сброс пароля; vSwitch и failover IP.
• Hetzner DNS Console — список зон, создание и редактирование записей, массовый импорт.
• Hetzner Object Storage — просмотр бакетов и объектов, загрузка и скачивание файлов, генерация подписанных ссылок.

Безопасность
• Никакого трекинга, никакой рекламы, никаких проприетарных зависимостей — 100% свободное ПО (GPL-3.0-or-later).
• Учётные данные хранятся в EncryptedSharedPreferences. Мастер-ключ привязан к Android Keystore (StrongBox при наличии) и защищён биометрией или экранной блокировкой.
• Только TLS 1.3 с certificate pinning для каждого эндпоинта Hetzner.
• FLAG_SECURE предотвращает утечку токенов через скриншоты и список недавних приложений.
• Автоматическое резервное копирование отключено — секреты никогда не покидают устройство.

Falco — это сообщество-проект, не связанный с Hetzner Online GmbH.
