Falco 1.1.0

Объединение проектов + глобальный поиск.

Главное:
- Настройки → Cloud → «Объединить все проекты»: списки (серверы,
  тома, сети, плавающие IP, файрволы, Storage Box, балансировщики,
  сертификаты, placement-группы, SSH-ключи, primary-IP)
  объединяются по всем проектам активного аккаунта. Выбор проекта
  скрывается.
- Перед каждым «+» (тома, сети, плавающие IP, сертификаты,
  placement-группы, SSH-ключи) показывается выбор проекта, чтобы
  заранее указать целевой проект в режиме объединения.
- Новый экран глобального поиска в боковом меню: при открытии
  параллельно индексирует все проекты, Robot-серверы и DNS-зоны,
  затем фильтрует локально при каждом нажатии. Касание результата
  автоматически переключает активный проект перед переходом на
  экран деталей.
- Артефакты сборки теперь называются `Falco-<version>-<type>.apk`.
