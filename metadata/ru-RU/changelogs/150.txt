Falco 1.5.0

Релиз с упором на безопасность и укрепление цепочки поставок. Два
раунда аудита прошлись по каждому потоку, работающему с секретами,
входами цепочки поставок или деструктивными действиями.

Главное:
- Пароли root, пароли rescue-режима и токены консоли теперь
  отображаются в общем одноразовом диалоге с FLAG_SECURE — ни
  скриншоты, ни предпросмотр в «Недавних» их не захватят.
- Чувствительные копии в буфер обмена устанавливают
  EXTRA_IS_SENSITIVE на Android 13+ и автоматически очищаются через
  60 с.
- TLS SPKI-пины обновлены по живой цепочке Hetzner; добавлены
  актуальные промежуточные Let's Encrypt (E7, R13).
- Валидация эндпоинта S3 переписана на okhttp HttpUrl: отвергает
  userinfo, нестандартные порты и любые path / query / fragment.
- Заголовок Idempotency-Key (UUIDv4) автоматически добавляется ко
  всем меняющим состояние вызовам Cloud- и Storage-Box-клиентов.
- toString() у DTO маскирует password и rootPassword у восьми
  затронутых Cloud- + Robot-DTO, чтобы секреты не попадали в логи
  или crash-репорты.
- Записи credential store теперь долговечны (commit на IO) — окно
  «фантомного аккаунта» после OOM посреди create() закрыто.
- Лист вставки SSH-ключа отклоняет блобы с маркерами приватного
  ключа.
- Импорт зон BIND ограничен 1 МиБ.
- Исключение rate-limit Robot теперь несёт заголовок Retry-After.
- Диалог «Поделиться» исключает приложения буфера обмена / заметок
  (MIUI, Samsung, Huawei) из чузера.

Сборка и CI:
- Каждое GitHub Action закреплено на 40-символьном SHA коммита.
- Контрольная сумма дистрибутива Gradle 8.11.1 зафиксирована.
- Права на keystore зафиксированы как 600 в release-сборке.
- Скачивание артефактов релиза отфильтровано по имени; глобы файлов
  релиза ограничены.
- Новый preflight: тег должен совпадать с versionName, иначе сборка
  падает.

Мелкие правки:
- Кнопка завершения мастера больше не прячется за панелью навигации.
- Тема по умолчанию переключена на светлую; перекрытие со статус-баром
  исправлено.
- MainActivity защищает ACTION_SECURITY_SETTINGS от отсутствующих
  обработчиков intent.
