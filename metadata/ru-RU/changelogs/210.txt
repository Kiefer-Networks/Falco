Falco 2.1.0

Major-version sweep across the build toolchain and the network /
storage stack. No behavioural changes for the user.

Toolchain:
- AGP 8.13.2 → 9.2.0; Hilt 2.58 → 2.59.2; Gradle 8.14.4 → 9.5.0.
- ProGuard `-keepattributes` widened to the explicit Runtime{Invisible,
  Visible}Annotations set required by R8 9.x.

Network:
- Retrofit 2.11.0 → 3.0.0 (forward binary-compatible).
- OkHttp 4.12.0 → 5.3.2. Every CertificatePinner / ConnectionSpec /
  Interceptor surface is preserved verbatim.

Storage (S3):
- MinIO 8.5.14 → 9.0.0. The 9.0 refactor renamed two classes Falco
  uses (Http.Method, DeleteRequest.Object) and inverted the
  `removeObjects` return semantics from successes to failures —
  S3Client now computes success-list = input - failures so callers
  see the same contract.

CI:
- All five GitHub Actions Dependabot major bumps merged
  (actions/checkout, setup-java, download-artifact,
  softprops/action-gh-release, gradle/actions).

Tests:
- PasswordRedactionTest extended to cover the privateKey + userData
  redactions added in v1.6.0.
