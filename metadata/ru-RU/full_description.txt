Falco — Android-клиент для сервисов Hetzner Online, уважающий
приватность. В одном защищённом приложении управляйте Hetzner Cloud,
выделенными серверами Robot и Storage Box, DNS-консолью и
S3-совместимым Object Storage.

Hetzner Cloud
- Серверы с метриками, питанием, сбросом пароля, ребилдом, сменой
  типа, ISO, rescue-режимом, бэкапами, консолью, обратным DNS,
  переименованием, защитой и удалением.
- Тома с мастером создания и живой оценкой ежемесячной цены,
  прикрепление/открепление, изменение размера, форматирование,
  automount.
- Сети: мастер, подсети, маршруты, смена IP-диапазона, экспорт
  маршрутов в vSwitch.
- Плавающие IP: назначить / снять, обратный DNS.
- Файрволы, балансировщики, сертификаты (загруженные или managed
  Let's Encrypt), placement-группы.
- SSH-ключи и Primary IP.
- Storage Box: снапшоты, субаккаунты, сброс пароля, переключатели
  доступа (SSH, Samba, WebDAV, ZFS).

Hetzner Robot
- Обзор выделенных серверов: статус, IP, ЦОД, оплачено до.
- Reset (sw/hw/ручной), Wake-on-LAN, rescue, обратный DNS, отмена
  заказа и её отзыв.
- CRUD vSwitch + прикрепление/открепление серверов.
- Маршрутизация failover-IP, SSH-ключи.

DNS-консоль
- Зоны: список, создание, редактирование, удаление.
- Записи: добавление, редактирование, удаление, массовое
  создание/обновление, импорт/экспорт BIND, валидация.
- Primary-серверы для secondary-зон.

Object Storage (S3-совместимый)
- Просмотр бакетов и префиксов, загрузка через foreground-сервис,
  скачивание по подписанной URL, копирование, пакетное удаление,
  предварительно подписанная URL для загрузки, версионирование.

Безопасность
- 100 % свободное ПО (GPL-3.0-or-later). Без Google Play Services,
  Firebase, аналитики и телеметрии.
- Учётные данные в EncryptedSharedPreferences с мастер-ключом,
  привязанным к Android Keystore (StrongBox при наличии).
- BiometricPrompt при каждом холодном старте и после настраиваемого
  тайм-аута в фоне.
- Только TLS 1.3 с pinning корней для всех четырёх API.
- FLAG_SECURE на каждом экране.

Внешний вид: Светлая / Тёмная / Системная / OLED-чёрная темы и пять
акцентных цветов (Hetzner красный, синий, зелёный, фиолетовый,
оранжевый). Переведено на английский, немецкий, испанский,
французский, итальянский, русский и упрощённый китайский.

Falco — общественное ПО, не связанное с Hetzner Online GmbH.
